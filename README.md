# summonAPI #
Based on modern world web MeanJS ( AngularJS, NodeJS, ExpressJS, MongoDB).
 
### Requirements- 
#### 
    npm@4.4.1
    node@v6.10.0
    mongo 
### Installations-
Clone the repository and enter the following commands in the terminal- 
####    
    cd summonAPI
    npm install   (this will install the package.json)
